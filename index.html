Collaboration 101

Users today are used to the realtime experience

Users expect online web tools to be collaborative like Notion, Figma and Docs. We are building an online web application at Slate, so we wanted realtime collaboration. We searched the web and didn't find a single tool that fitted our needs and was simple to use.
We found out that realtime solutions are hard to implement correctly, and there is no "adding it later" without much time cost. So we decided we needed to create something new from scratch and since we are already doing it, why not spend a little extra time on it and make it reusable for anyone. After a lot of research and many many versions and rewrites, a product was born that we are very proud of and that we think is going to have a great impact on the implementation of realtime collaboration.
